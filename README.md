# Flood Monitoring System using ESP32 (IIoT)

## Description
This project simulates an Industrial Internet of Things (IIoT) Flood Monitoring System using an ESP32 and an HC-SR04 ultrasonic sensor. The system monitors water level, activates visual and audible alerts during flood conditions, and uploads sensor data to ThingSpeak for real-time cloud monitoring.

## Components
- ESP32
- HC-SR04 Ultrasonic Sensor
- Green LED
- Red LED
- Buzzer
- ThingSpeak
- Wokwi

## Software Used
- Arduino IDE
- Wokwi Simulator
- ThingSpeak
- GitHub

## Features
- Water level monitoring
- Flood warning detection
- LED and buzzer alerts
- Cloud data logging using ThingSpeak
- Fully simulated without physical hardware
